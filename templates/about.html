About

This is an attempt to recreate a simpler lighter clone of Crunchbase. Some of the Data used is real data gotten from using the free curnchbase basic API, and the rest of the data is randomly generated.

Data such as employee and funding information is randomly generated, while data such as description and Social Media details are gotten from Crunchbase.

This app is strictly for demo purposes. I will not be held liable for any misinformation regarding any of the companies searched.
